# Photo Viewer

This repository contains a simple photo viewer web application built with HTML, CSS, and JavaScript. This was completed for CMSC335 Web Application Developmetn Course at UMD.

## Features

* Display and cycle through photos
* Move to the next, previous, first, or last image
* Load images from a local folder
* Load image data from a JSON URL
* Run a regular slideshow
* Run a random slideshow
* Stop the slideshow
* Reset form values

## Technologies

* HTML
* CSS
* JavaScript

## Project Note

This project was completed as a coursework assignment. It helped practice DOM manipulation, event handling, form inputs, image loading, and basic slideshow functionality.
